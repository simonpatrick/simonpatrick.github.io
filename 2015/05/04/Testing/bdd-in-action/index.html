BDD是什么（Behavior Development Driven）

BDD is using example in multiple levels to create a shared understanding and suface uncertainty
to deliver software that matters.

example 和 shared understanding是关键字。BDD的目的就是通过几个示例来创建出对于交付软件的一致认识。

TDD 和 BDD

TDD: Test Driven Development

  • TDD is not about writing tests
  • TDD is a design strategy:
    • Write better-design code(写更好设计的代码）
    • Hava more confidence in our code（对自己的代码更自信）
    • Make changes more easily（更容易的改动）
    • Write Code that meets user requirements more accurately
    • Build up a comprehensive set to automation suites
