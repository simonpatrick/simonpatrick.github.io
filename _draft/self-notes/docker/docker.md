### Docker
#### Docker 概念
##### Docker 镜像
镜像只是一个模板，可以通过模板创建docker容器
##### Docker 容器
容器从镜像创建运行实例，可以启动，开始，停止，删除。同时又相互隔离
镜像是只读的，容器创建一层可作为最上层

##### Docker 仓库
仓库和仓库注册器（Registry），仓库注册器上可能有很多个不同的仓库
仓库:

* [Docker Hub](https://hub.docker.com/)
* [Docker Pool](http://www.dockerpool.com/)
* [Docker CN](https://docker.cn/)
