# HEXO 

## HEXO Installation

```shell
npm install -g hexo-cli
```

## HEXO start

```shell
hexo init && npm install
```

## HEXO 目录结构

- _config.yml
- package.json
- scaffolds: 文档模板
- scripts: 脚本扩展
- source：资源文件，主要指文档
- themes：主题位置
