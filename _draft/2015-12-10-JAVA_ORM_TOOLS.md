# JAVA ORM tools

从以下几个角度比较Java ORM tools:
- Learning Curve/Ease of Use
- Productivity
- Maintainability/Stability
- Performance/Scalability
- Ease of Troubleshooting

## ORM tools的用途和常见优缺点
- ORM for transactions
- straightforward CRUD and SQL/thin DAL for reporting
- ORM工具一般不快
- XML 配置多
