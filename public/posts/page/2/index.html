SQL 注入例子 一下例子是有SQL注入漏洞的一段程序： conn = pool.getConnection( ); String sql = "select * from user where username='" + username +"' and password='" + password + "'"; stmt = conn.createStatement(); rs = stmt.executeQuery(sql); if (rs.next()) { loggedIn = true; out.println("Successfully logged in"); } else { out.println("Username and/or password not recognized"); } 由于SQL是拼接的，所有如果用户直接食用username，或者password中 使用了如: admin' OR '1'='1 最后执行的SQL就是： select * from user where username='admin' OR '1'='1' and password=' ' 明显，这样就会把所有的用户信息拿到了 防范措施 Defense Strategy prevent SQL injection: All queries should be parametrized.
